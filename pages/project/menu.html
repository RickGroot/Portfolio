Pure CSS menu| Code | Live

Deze menukaart is gemaakt door middel van pure CSS, er wordt nergens gebruik gemaakt van JavaScript en andere plugins. Doordat alles gemaakt is in CSS is het geen probleem wanneer de gebruiker geen JavaScript aan heeft staan in de browser. De vorm en inhoud versterken elkaar door een sterke huisstijl, waarbij alleen gebruik is gemaakt van 2 kleuren. Het prototype werkt het best op chromium browsers.

Print stylesheet

Omdat dit een menukaart is zit er ook een print stylesheet ingebouwd. Wanneer de html pagina wordt afgedrukt verschijnt er een mooi afgewerkte print, dat het restaurant goed kan gebruiken. De menukaart kan dus makkelijk worden aangepast bij veranderingen en kan vervolgens meteen worden uitgeprint.

Andere bijzonderheden

Voor extra uitdaging is deze menukaart volledig responsive gemaakt. Het responsive design is gemaakt zonder media queries. Ook maakt het hele project geen gebruik van classes in de HTML, alles is gestyled door alternatieve CSS selectors.
